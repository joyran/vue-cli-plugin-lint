# vue-cli-plugin-lint

vue cli 自定义脚手架

## 特性

脚手架默认开启如下功能

- less
- router
- vuex
- eslint
- stylelint         css 检查
- prettier          代码美化
- commitlint        commit message 校验
- commitizen        commit message 校验
- lint-staged       commit 前代码检查和美化
- standard-version  自动打 tag 和生成 CHANGELOG.md

## 使用

```
vue create --preset joyran/vue-cli-plugin-lint my-app
```
